# Gemini 3 Pro - Vibe Code Competition

Kaggle competition: https://www.kaggle.com/competitions/gemini-3

## Competition

- **Dates**: December 5-12, 2025 (1 week)
- **Prize**: $500,000 in Gemini API credits
- **Stack**: FastAPI + SvelteKit

## Judging

| Criteria | Weight |
|----------|--------|
| Real-world impact | 40% |
| Gemini 3 Pro capabilities | 30% |
| Creativity | 20% |
| Presentation (2-min video) | 10% |

## Each Project Contains

- `spec.md` - Full specification (features, architecture, API design)
- `plan.md` - Implementation plan with AI Studio prompts

## How to Use

1. Open [Google AI Studio](https://aistudio.google.com/)
2. Select **Gemini 3 Pro**
3. Copy prompts from project's `plan.md`
4. Generate code and iterate

## Quick Start

Pick a project and use the prompts in `plan.md` to generate:
- Backend: FastAPI application
- Frontend: SvelteKit application
